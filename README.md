# Derk Ottersberg

Embedded developer focused on microcontrollers, hardware systems, and low-level software.

I work across the full stack when needed, but my primary interest is building reliable embedded systems that interact with real hardware. My projects often combine firmware, electronics, backend services, and custom tools.

I also build experimental projects such as game mods and educational reverse-engineering tools to better understand how complex software systems work.
